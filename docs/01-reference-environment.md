# Reference Environment

This project uses a simulated financial-services company with approximately 500 employees and 650 managed endpoints.

## Identity and Cloud

* Microsoft Entra ID
* Hybrid identity with on-premises Active Directory
* Microsoft 365
* Exchange Online
* SharePoint Online
* OneDrive
* Microsoft Teams

## Endpoints

* Windows 11 workstations
* Windows Server
* Limited number of Linux servers
* Microsoft Intune
* Microsoft Defender for Endpoint

## Network

* Two office locations
* Remote workforce
* Palo Alto perimeter firewall
* GlobalProtect VPN
* Internal DNS and DHCP
* Separate user, server, and administrative network segments

## Additional Infrastructure

* Small AWS environment
* Internet-facing web applications
* Third-party SaaS services

## Security Operations

* Elastic-based SIEM
* Centralized ticketing and case management
* n8n for selected SOC automations

All systems, users, events, and data used in this project are fictional or synthetic.
