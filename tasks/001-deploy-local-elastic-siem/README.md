# Task 001: Deploy a Local Elastic SIEM Environment

## Objective

Deploy a local Elastic environment that will be used for telemetry ingestion, detection testing, investigation, and dashboard development throughout the project.

## Requirements

- Elasticsearch
- Kibana
- Docker-based deployment
- Persistent storage
- Authentication enabled
- Accessible Kibana interface
- Documented startup and shutdown process

## Acceptance Criteria

- Elasticsearch is running and reports a healthy cluster status.
- Kibana is accessible from the local machine.
- Kibana can communicate with Elasticsearch.
- Data persists after the containers are restarted.
- No credentials or secrets are committed to the repository.
- Deployment and validation steps are documented.
