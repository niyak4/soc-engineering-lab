# soc-engineering-lab

A hands-on SOC engineering lab focused on telemetry onboarding, detection engineering, SIEM, automation, and incident response.

## Project Scope

This repository documents and implements a simulated SOC engineering environment.

The project will cover SIEM architecture, telemetry onboarding, data normalization, detection engineering, investigation workflows, and SOC automation.

The first stage focuses on defining the reference environment and designing a practical SIEM telemetry onboarding strategy.

Only synthetic or sanitized data will be used.
